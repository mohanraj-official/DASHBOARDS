# Columns

| Column Name          | Description                  |
| -------------------- | ---------------------------- |
| track_name           | Name of the song             |
| artist(s)_name       | Artist name                  |
| artist_count         | Number of artists involved   |
| released_year        | Year of release              |
| released_month       | Month of release             |
| released_day         | Day of release               |
| in_spotify_playlists | Count of Spotify playlists   |
| streams              | Total Spotify streams        |
| bpm                  | Beats per minute             |
| key                  | Musical key                  |
| mode                 | Musical mode, Major or Minor |
| danceability_%       | Danceability score           |
| valence_%            | Positivity level             |
| energy_%             | Energy level                 |
| acousticness_%       | Acoustic score               |
| instrumentalness_%   | Instrumental presence        |
| liveness_%           | Live performance likelihood  |
| speechiness_%        | Spoken words presence        |

# Summary Values and Key Findings

KPI summary from the dashboard:

* Total songs analysed: 952
* Unique artists: Multiple global artists
* Average BPM range: Mid-tempo dominant
* Majority mode: Major
* Artist collaborations mostly single-artist tracks

<BR>

# Insights

* Most songs fall under Major mode, indicating upbeat composition trends.
* A small group of artists contribute a high volume of tracks.
* Single-artist songs dominate the dataset over collaborations.
* BPM values cluster around mid-tempo ranges, suitable for mainstream playlists.
* Recent years show higher streaming volumes compared to earlier years.
